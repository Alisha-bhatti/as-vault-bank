# 🏛 AS-Vault Bank

A full-stack banking web application with a beautiful advertisement-style landing page.

## 🌐 Live Demo
👉 **[Click here to open AS-Vault Bank](https://alisha-bhatti.github.io/as-vault-bank)**

## ✨ Features
- Beautiful landing page with animated bank building illustration
- PIN-secured account login
- Deposit & Withdraw money
- Transaction history
- Interest Calculator (Simple & Compound)
- Account deletion
- Real backend with Flask + SQLite database

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript — hosted on GitHub Pages
- **Backend:** Python Flask + SQLite — hosted on Render
- **Original project:** Java CLI Banking App (converted to full stack)

## 📁 Files
- `index.html` — Frontend
- `app.py` — Flask backend API
- `requirements.txt` — Python dependencies
- `database.db` — SQLite database
